# **函数详解**
这段代码的功能是用于检测手牌中成对的牌并进行消除。这里特意提一嘴这段代码主要是因为它使用了collections包，这个Python内置包中的很多函数在解决算法问题、数据处理时很有用处，也很方便。  
**当然，仅在做一些简单处理时或使用纯净版Python时很好用，不讨论使用numpy或者pandas的情况，个人见解，切勿上纲上线。**
```
from collections import Counter

def card_remove(card_list):
    card_num = Counter(card_list)
    new_card_list = [card for card in card_num.keys() if card_num[card] == 1]
    return new_card_list
```
这里的Counter输出了一个键为列表中的元素，值为元素对应的出现次数的字典。
